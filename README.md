# nuxt-starter

这是一个 `Nuxt3` + `TailwindCSS` + `TypeScript` + `ESLint` + `Prettier` 的起手式

这个工程的搭建笔记可以[在这里查看](https://welives.github.io/blog/front-end/engineering/nuxt.html)

## 使用

```sh
pnpm install
```

### 开发模式

```sh
pnpm dev
```

### 打包

```sh
pnpm build
```

### 预览

```sh
pnpm preview
```
