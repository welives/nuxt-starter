# ts-vant-starter

这是一个 `Vant` + `TailwindCSS` + `TypeScript` + `ESLint` + `Prettier` 的起手式

`main`分支是以 Vite 为基础搭建的，`with-nuxt3`分支是以 Nuxt3 为基础搭建的

## 使用

```sh
pnpm install
```

### 开发模式

```sh
pnpm dev
```

### 打包

```sh
pnpm build
```

### 预览

```sh
pnpm preview
```
